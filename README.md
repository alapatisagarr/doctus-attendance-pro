# Doctus Attendance Pro

A modern employee attendance management system built with React 19, Vite, Tailwind CSS, React Router, Firebase auth, Firestore-ready local storage data model, Chart.js, XLSX, jsPDF, and React Icons.

## Features
- Glassmorphism HRMS UI
- Protected routes and login experience
- Employee CRUD
- Attendance CRUD and export
- Leave management
- Reports and charts
- Settings and holiday-ready configuration
- Netlify-ready SPA routing

## Development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```
